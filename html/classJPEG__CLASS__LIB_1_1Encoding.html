static byte [] JPEG_CLASS_LIB.Encoding.EncodeAC ( List< short[]>  data)
inlinestatic

Вычисляет коды для AC коэффициентов (F.1.2.2.1). Для каждого ненулевого AC генерируется составной байт, RRRRSSSS (2 по 4 бита). SSSS - категория для AC коэффициента, RRRR - смещение текущего коэффициента относительно предыдущего (число нулей между предыдущим и текущим ненулевыми AC). Если число нулей больше 15, генерируется код 0xF0 (15 нулей). Если все оставшиеся коэффициенты равны нулю, то генерируется код 0x00 (конец блока). Все сгенерированные коды идут последовательно для всех блоков.

Аргументы
dataСписок блоков 8x8 после DCT и обхода зигзагом.
Возвращает
Массив последовательных кодов для AC коэффициентов всех блоков
